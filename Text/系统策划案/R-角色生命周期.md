### 角色生命周期系统与交互设计

#### 系统概述
角色生命周期系统模拟角色从出生到死亡的全过程，强调"人性社会化+阶段演化"的方法。系统聚焦五个核心社会化阶段：家庭社会化（0-15岁）、同伴社会化（15-25岁）、职业社会化（25-40岁）、权力社会化（40-60岁）、传承社会化（60岁+），并可选进入"死亡与传承"阶段。生命周期深度整合人性考验、试探机制、阵营对抗、道德发展、命运因果等系统，每个阶段通过关键人性测试事件推动角色变化，形成完整的人性发展轨迹和可追溯的道德传记。

- **生命周期结构**：每个角色生命周期为结构体：
  {
    角色ID,
    当前社会化阶段,
    基因模板ID,
    天赋ID[],
    人性特质矩阵,
    道德发展阶段,
    阵营倾向记录,
    人性考验历史[],
    试探行为档案[],
    社会化事件列表[],
    道德选择记录[],
    传承记录ID[],
    死因与道德评价（可为空）
  }
- **交互方式**：通过"人性社会化事件系统"推动角色演化。
  示例：
  - 家庭社会化阶段 → 触发"孝顺vs自我"人性考验 → 影响"道德基础"和"阵营倾向"；
  - 权力社会化阶段 → 参与"权力腐蚀试探" → 获得"权谋者"或"清廉者"标签。
- **数值表**：
  - `社会化阶段表`：家庭社会化（道德启蒙、价值观植入）、同伴社会化（价值观冲突、友情考验）、职业社会化（责任vs私情）、权力社会化（权力vs道德）、传承社会化（智慧传承、最终考验）。
  - `人性考验权重矩阵`：定义不同社会化阶段的人性测试类型、触发概率、影响权重（如青少年期友情背叛影响+30%，权力期腐败诱惑影响+50%）。
  - `道德发展概率表`：计算关键人性选择对道德阶段进展的影响（如坚持正义+20%道德进展，妥协现实-10%道德退化）。
- **行为树设计**：每个角色绑定人性驱动的行为树模板（如"理想主义型""现实主义型""权谋试探型"），通过社会化阶段和人性考验结果输入决定分支，与个性系统和阵营系统联动。
- **状态机设计**：
  - 状态枚举如：道德纯真、价值观形成、社会化适应、权力考验、道德升华、传承智慧、道德评判。
  - 触发器：社会化阶段转换、重大人性考验、道德冲突事件、阵营压力、试探行为后果。
  - 状态流转：如"道德纯真"→"价值观冲突"（同伴影响）→"社会化适应"（职业要求）→"权力考验"（诱惑测试）→"道德升华"（智慧觉醒）→"传承智慧"（经验传授）→"道德评判"（历史定位）。
- **人性特质模型**：基于道德发展理论的多维特质系统
  - 道德敏感度（0~100）：对道德问题的感知能力
  - 价值观坚定度（0~100）：在压力下坚持价值观的能力
  - 试探敏感度（0~100）：察觉他人试探意图的能力
  - 阵营忠诚度（0~100）：对所属阵营的忠诚程度
  - 人性复杂度（0~100）：处理复杂人性问题的智慧
  - 每维范围：0~100，初始由基因+家庭教育决定，后续通过人性考验调整。
- **流程概览**：
  1. 出生 → 父母基因决定人性特质基础+家族道德传统植入；
  2. 家庭社会化 → 道德启蒙事件塑造价值观基础+初始阵营倾向；
  3. 同伴社会化 → 价值观冲突事件+友情忠诚考验+试探机制启动；
  4. 职业社会化 → 责任vs私情的人性考验+阵营选择压力；
  5. 权力社会化 → 权力腐蚀考验+重大道德抉择+阵营对抗激化；
  6. 传承社会化 → 人生智慧总结+道德传承+最终人性考验；
  7. 死亡与传承 → 道德评价生成、人性档案传承、价值观影响后代。
  每个节点均写入"人性发展档案"，用于道德评价、命运因果构建或人性教育生成。

#### 详细机制
- **出生与基因传承**：从父母基因库和道德业力记录生成，决定外貌、天赋、人性特质基础、道德敏感度（链接命运系统的业力传承）。文化/生态影响（如儒家文化区域道德敏感度+10%）。
- **基因与人性整合系统**：决定外貌（物理特征）、天赋（技能倾向）、人性特质（道德发展潜力）、命运倾向（善恶报应敏感度，与M-命运.md整合）。基因与人性整合：基因模板定义基础人性特质ID，人性特质影响道德发展轨迹（如高道德敏感度增加正义事件概率）。道德业力作为命运系统的初始片段，由父母道德选择历史权重计算。
- **家庭社会化阶段**（0-15岁）：道德启蒙（价值观基础形成），家族传统学习（阵营倾向植入），长辈榜样模仿（行为模式建立），初次人性考验（孝顺vs自我、服从vs反抗）。
- **同伴社会化阶段**（15-25岁）：价值观冲突处理（不同背景朋友的影响），友情忠诚考验（友情vs利益、从众vs独立），试探机制启动（开始学会试探他人和被试探），初步阵营选择（基于同伴圈子的政治倾向）。
- **职业社会化阶段**（25-40岁）：职业道德考验（职责vs私情、规则vs变通），社会地位确立（影响阵营选择和权力欲望），权力初体验（小范围权力使用的道德考验），家庭责任平衡（个人理想vs家庭责任）。
- **权力社会化阶段**（40-60岁）：重大权力考验（权力vs道德、传统vs革新），阵营对抗激化（政治立场的最终确立），道德底线测试（极限情况下的选择），影响力扩大（对他人和社会的责任感）。
- **传承社会化阶段**（60岁+）：人生智慧总结（对一生选择的反思），道德传承责任（向后代传授人生智慧），最终人性考验（面对死亡时的道德坚守），历史地位确立（社会对其道德评价的形成）。

#### 人性考验事件系统
**家庭社会化阶段的人性考验**：
- 孝顺vs自我考验：父母要求与个人意愿冲突时的选择
- 服从vs反抗考验：面对不合理权威时的态度
- 分享vs独占考验：资源稀缺时是否愿意与他人分享
- 诚实vs谎言考验：说谎能获得好处时是否坚持诚实

**同伴社会化阶段的人性考验**：
- 友情vs利益考验：朋友利益与个人利益冲突时的选择
- 从众vs独立考验：群体压力下是否坚持个人判断
- 忠诚vs背叛考验：朋友犯错时是否选择包庇
- 竞争vs合作考验：在竞争环境中是否选择公平竞争

**职业社会化阶段的人性考验**：
- 职责vs私情考验：公务与私人关系冲突时的选择
- 规则vs变通考验：严格执行规则还是灵活处理
- 上级vs下属考验：上级错误决策时是否敢于反对
- 短期vs长期考验：短期利益与长期价值的权衡

**权力社会化阶段的人性考验**：
- 权力vs道德考验：权力能解决问题但违背道德时的选择
- 传统vs革新考验：维护传统还是推动变革
- 个人vs集体考验：个人理想与集体利益的冲突
- 现实vs理想考验：理想主义与现实主义的平衡

**传承社会化阶段的人性考验**：
- 真相vs和谐考验：是否向后代说出人生的全部真相
- 传统vs创新考验：传承传统价值观还是鼓励创新思维
- 宽恕vs正义考验：对过往恩怨的最终态度
- 执着vs放下考验：对人生成就和遗憾的最终态度

#### 试探机制在生命周期中的演化
**试探能力发展轨迹**：
- 家庭社会化：被动接受试探，学习基本的人际判断
- 同伴社会化：开始主动试探他人，学习社交技巧
- 职业社会化：试探技巧成熟，用于职场竞争和合作
- 权力社会化：试探成为权力博弈的重要工具
- 传承社会化：试探智慧达到巅峰，能够识破一切试探

**反试探能力发展**：
- 早期：容易被试探，缺乏防范意识
- 中期：开始察觉试探，但反应不够成熟
- 成熟期：能够识别大部分试探，并采取适当对策
- 智慧期：不仅能识破试探，还能反向利用试探获取信息
- 超脱期：超越试探博弈，以更高的智慧处理人际关系

#### 阵营发展轨迹
**阵营认同的演化过程**：
- 家庭植入期：接受家族传统的阵营倾向
- 同伴影响期：受朋友圈子影响，可能产生阵营动摇
- 职业选择期：职业要求与个人倾向的冲突和调和
- 权力确立期：在权力博弈中确立最终的阵营立场
- 超越整合期：可能超越阵营局限，形成更高层次的价值观

**阵营转换的触发条件**：
- 重大价值观冲击：个人经历与原有阵营理念的根本冲突
- 利益驱动转换：新阵营提供更好的发展机会
- 情感因素影响：重要关系人的影响和劝说
- 道德觉醒转换：通过人性考验获得新的道德认知
- 现实压力迫使：生存环境变化导致的被迫选择

#### 道德发展阶段理论整合
**基于科尔伯格道德发展理论的六阶段模型**：
1. **惩罚服从阶段**：避免惩罚，服从权威
2. **相对功利阶段**：满足个人需要，偶尔满足他人需要
3. **寻求认可阶段**：寻求他人认可，做"好孩子"
4. **维护权威阶段**：维护社会秩序，履行义务
5. **社会契约阶段**：理解规则的相对性，重视程序正义
6. **普遍伦理阶段**：基于普遍的伦理原则做出判断

**道德发展的影响因素**：
- 认知发展水平：智慧修为影响道德推理能力
- 社会经验丰富度：人生阅历影响道德判断的复杂性
- 文化环境影响：不同文化背景提供不同的道德框架
- 人性考验强度：重大道德冲击促进道德发展跃迁
- 榜样示范作用：重要他人的道德行为影响个人发展

#### 传承机制的深度设计
**道德传承系统**：
- 价值观传承：核心价值观念的代际传递
- 行为模式传承：处理道德冲突的方式方法
- 人际智慧传承：试探与反试探的经验技巧
- 阵营关系传承：政治立场和社会关系网络
- 道德债务传承：前代未完成的道德责任

**传承效果的影响因素**：
- 血缘关系亲密度：直系血亲传承效果最强
- 情感联系深度：感情深厚的关系传承效果更好
- 价值观契合度：价值观相近的传承更容易成功
- 传承者道德威望：德高望重者的传承影响力更大
- 接受者开放程度：开放性高的角色更容易接受传承

**传承失败的可能性**：
- 代际价值观冲突：新旧价值观的根本分歧
- 传承者道德污点：前代的道德问题影响传承效果
- 社会环境变化：时代变迁导致传承内容过时
- 接受者主观拒绝：后代主动拒绝前代的价值观
- 外部势力干预：其他阵营的影响和拉拢

通过这种深度整合的角色生命周期系统设计，每个角色的一生都成为一个完整的人性发展历程，体现了"修身五境"和"善恶报应"的核心理念，同时为玩家提供了深刻的人性思考和道德成长体验。
