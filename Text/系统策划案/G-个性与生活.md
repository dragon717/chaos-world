### 个性与生活系统与交互设计

#### 系统概述

**基于"事实—观点—立场—信仰"四层人际关系架构的个性与生活系统**

个性与生活系统是角色人性探索和社会化进程的核心载体，通过深度模拟角色的内在特质、情感状态、日常行为和社会互动，在"事实—观点—立场—信仰"四个层次上展现角色的人性复杂性和社会化发展。系统以"人性测试+社会化进程"为核心理念，通过"个性发展+生活体验+人际互动"三重机制，全面考验角色在日常生活中的人性品质和社会适应能力。

系统通过深度模拟角色的内在特质、情感状态、日常行为和社会互动，在四个层次上展现人性的复杂性和社会化的渐进性。

**四层架构在个性与生活中的体现：**

🎭 **事实层（生活实践档案）**：
- **日常行为档案**：记录角色在日常生活、社会交往、道德选择中的具体行为和表现
- **人际互动档案**：记录角色与他人的交往过程、关系发展、互动模式等客观情况
- **社会化档案**：记录角色的社会适应过程、角色转换、身份认同等具体历程
- **人性测试档案**：记录角色在各种人性考验、道德冲突中的具体选择和反应

🧠 **观点层（个性认知演化）**：
- **自我认知演化**：角色对自身性格、能力、价值的认知理解逐步深化
- **他人认知演化**：角色对他人性格、动机、可信度的认知观念不断发展
- **社会认知演化**：角色对社会规则、人际关系、社会现象的认知评价持续演进
- **生活认知演化**：角色对生活意义、人生目标、幸福标准的认知逐渐成熟

⚖️ **立场层（个性身份选择）**：
- **理想主义者身份**：选择坚持理想和原则的生活方式，重视精神追求和道德坚守
- **现实主义者身份**：选择适应现实和环境的处世态度，重视实用性和生存智慧
- **社交活跃者身份**：选择积极参与社会活动的生活模式，重视人际关系和社会影响
- **独立自主者身份**：选择独立自主的生活方式，重视个人自由和独特性

🔥 **信仰层（个性核心信念）**：
- **人性本善信念**：坚信人性本质是善良的，相信通过教育和感化可以改变他人
- **自我实现信念**：坚信每个人都有实现自我价值的权利和能力，追求个人成长
- **社会和谐信念**：坚信社会和谐是最高价值，愿意为集体利益牺牲个人利益
- **真实自我信念**：坚信保持真实自我是最重要的，不愿为迎合他人而改变自己

通过"个性发展+生活体验+人际互动"的三重机制，在四层架构中全面考验角色的人性品质和社会智慧。

系统强调"人性测试"与"试探机制"在四个层次上的有机结合，让每个角色在面对道德选择、利益冲突、情感纠葛时，从事实记录到观点形成，从立场选择到信仰坚持，全方位展现真实的人性光辉与阴暗。

#### 系统目标与玩法机制

| 系统目的 | 对应玩法机制 | 玩家行为体现 |
|---------|-------------|-------------|
| 呈现个性差异 | 个性影响技能效果、交互反应 | 选择性格影响队伍配置 |
| 推动剧情演进 | 人性测试与状态变迁影响支线和结局 | 道德选择改变故事走向 |
| 强化互动张力 | 试探机制反馈行为+关系改变 | 对话分支、关系值浮动 |
| 体现角色成长 | 社会化阶段推动系统更新 | 随时间解锁新事件、新技能 |
| 阵营动态演化 | 阵营倾向引发冲突与合作 | 阵营切换改变资源与剧情 |

#### 核心机制设计

**1. 个性系统机制 → "角色人格模板 + 战斗被动 + 社交反馈"**

每个角色绑定一套"人格模板"，影响其：
- 战斗技能被动（如"理性型"可减免混乱状态，"情感型"暴击更高）
- 社交偏好（"独立型"更难提高亲密度，"从众型"更容易被操控）
- 与阵营关系（"传统型"更亲近保守派，"革新型"更亲技术派）

**个性结构**：每个"个性"记录为结构体：{个性ID、人性特质矩阵、社会化阶段、试探敏感度、阵营倾向、道德底线、情感状态、战斗被动效果、社交反应模式、人性测试记录}，储存在角色个人档案中。

**2. 人性测试系统 → "事件驱动的道德分支+状态切换"**

触发"人性测试场景"后进入多选项事件界面：
- 每次选择带有颜色标记（理想主义、实用主义、利他、自利）
- 选项影响当前人格状态（如："道德坚定"→"价值冲突"）
- 决策会开启不同支线剧情，或影响队友忠诚度/阵营立场

**3. 试探机制 → "社交战斗 + 情绪状态 + 反试探玩法"**

借鉴《女神异闻录》《Disco Elysium》的心理战机制：
- 社交试探阶段：对话选项含隐藏试探，如"你觉得老大是个怎样的人？"
- 角色敏感度影响反应：高敏角色会警觉并启动反试探策略，如回避、反问
- 失败反馈：误判可能引发信任度下降、关系破裂、信息误导

**4. 行为树与状态机 → "AI角色反应+剧情自动演进"**
- 行为树：根据人格矩阵 + 当前状态，决定非玩家角色（NPC）主动行为
- 状态机：控制NPC的心理状态变化（如："社会适应"→"利益背叛"）

实现NPC "自己演戏"，玩家只是卷入其中。
#### 战斗与剧情联动的动态反馈系统

| 战斗维度 | 对应的人格反馈 |
|---------|---------------|
| 战斗中做出选择 | 战术倾向反映人性，如放弃救人选择会被记录 |
| 同伴忠诚/阵营变动 | 人性测试结果影响战斗中同伴行为，如反叛、迟钝、献祭 |
| 情绪状态系统 | 情绪如"内疚""愤怒"影响技能效果（暴击率、行动力等） |
| 阵营立场转变 | 选择新阵营后，获得新阵营专属技能或装备，失去旧有资源 |

#### 模块化配置表设计

为便于开发，系统采用模块化配置表结构：

| 模块 | 示例字段 |
|------|----------|
| 个性配置表 | 人格ID、特质值（理想主义度、依附度等）、战斗被动、交互特征 |
| 社会化阶段表 | 年龄段、阶段影响因子、可解锁事件、人格变化方向 |
| 人性测试事件表 | 事件ID、触发条件、选项列表、影响字段（状态、人性值、信任度） |
| 试探交互表 | 台词、目标对象、敏感度要求、成功判定方式、后续关系影响 |
| 阵营倾向表 | 阵营ID、偏好人格、冲突阵营、转换代价、阵营技能 |

#### 数值表详细设计

- `人性特质表`：生存本能vs道德理想、个人野心vs集体利益、传统守护vs变革追求、情感依恋vs理性判断，影响关键选择和阵营倾向。
- `社会化影响矩阵`：地域文化、阶层差异、宗教信仰、历史事件对个性塑造的权重影响。
- `试探反应表`：不同个性在面对言语试探、行为试探、利益试探、危机试探时的反应模式。
- `道德测试表`：饥荒选择、权力诱惑、忠诚考验、技术伦理等场景下的选择倾向。

**行为树设计**：每个角色绑定行为树模板（如"理想主义型""现实主义型""机会主义型"），通过人性测试结果和社会化阶段输入当前状态，决定执行分支。链接生命周期：行为树在不同社会化阶段体现不同的价值观和行为模式。
- **状态机设计**：
  - 状态枚举如：道德坚定、价值冲突、利益权衡、情感纠结、社会适应、阵营选择。
  - 触发器：人性测试事件、试探行为感知、社会化压力、阵营冲突、道德底线触碰。
  - 状态流转：如"价值冲突"→"内心挣扎"→"道德选择"→"行为调整"→"社会适应"。
- **流程概览**：
  1. 系统监测角色社会化阶段和人性特质；
  2. 触发人性测试场景或试探事件；
  3. 进入相应心理状态；
  4. 执行选择响应（道德决策、行为表现等）；
  5. 更新人性档案和社会化进程，影响阵营倾向和未来选择。

#### 与角色生命周期的整合
- **家庭社会化阶段**（0-15岁）：基因决定人性特质基础，家族价值观植入形成道德框架，教育事件塑造初始阵营倾向。人性测试：孝顺vs自我、服从vs反抗。
- **同伴社会化阶段**（15-25岁）：朋友圈影响价值观重塑，经历价值观冲突与选择，试探机制开始活跃。人性测试：友情vs利益、从众vs独立。
- **职业社会化阶段**（25-40岁）：职业角色要求调整行为模式，社会地位确立影响阵营选择，权力诱惑考验道德底线。人性测试：职责vs私情、规则vs变通。
- **权力社会化阶段**（40-60岁）：承担更大社会责任，影响力扩大带来更多试探，阵营对抗激化。人性测试：权力vs道德、传统vs革新。
- **传承社会化阶段**（60岁+）：经验传授与价值观传承，人性智慧达到巅峰，成为他人的试探对象和道德标杆。
- **死亡与传承**：人性特质、社会化经验、道德选择记录部分传承给后代，影响新角色的初始倾向和阵营预设。

#### 人性测试场景系统
**饥荒选择场景**：
- 触发条件：资源稀缺、灾难事件、战争影响
- 测试内容：优先照顾老弱还是壮年劳力？个人储备还是集体分享？
- 影响因素：个人野心vs集体利益特质、地域文化背景、宗教信仰
- 结果影响：道德声望变化、阵营倾向调整、后续事件概率

**权力诱惑场景**：
- 触发条件：获得重要情报、职位晋升机会、政治选择
- 测试内容：上报朝廷获得赏赐还是暗中利用？忠诚还是背叛？
- 影响因素：生存本能vs道德理想特质、当前社会化阶段、阵营压力
- 结果影响：政治地位变化、信任度调整、敌友关系转换

**忠诚考验场景**：
- 触发条件：发现师父/朋友叛国、利益冲突、道德两难
- 测试内容：举报还是包庇？大义还是私情？
- 影响因素：情感依恋vs理性判断特质、关系深度、社会压力
- 结果影响：关系网络重构、心理创伤、价值观重塑

**技术伦理场景**：
- 触发条件：掌握先进技术、技术泄露风险、创新机会
- 测试内容：独占还是分享？保密还是公开？传统还是革新？
- 影响因素：传统守护vs变革追求特质、技术阵营、国家利益
- 结果影响：技术发展方向、阵营关系、历史进程

#### 试探机制深度设计
**试探敏感度系统**：
- 高敏感度：能快速察觉他人试探意图，采取相应对策
- 中敏感度：部分察觉试探，反应较为迟缓
- 低敏感度：难以察觉试探，容易被他人操控
- 影响因素：智慧修为、社会化经验、人际关系复杂度

**反试探策略**：
- 信息控制：选择性透露信息，保持神秘感
- 反向试探：通过反问和观察获取对方信息
- 虚假信息：故意透露错误信息，误导对方
- 直接对抗：公开拒绝试探，展现强硬态度

#### 阵营倾向动态系统
**阵营倾向值**：
- 朝廷保守派倾向：维护传统、服从权威、重视礼法
- 朝廷革新派倾向：推行变法、技术创新、效率优先
- 地方豪强派倾向：地方利益、民众基础、实用主义
- 民间义士派倾向：正义理想、民众福祉、道德优先
- 技术中立派倾向：技术发展、知识传承、超越政治
- 宗教超脱派倾向：精神追求、道德完善、超越世俗

**倾向转换机制**：
- 价值观冲突：个人理念与当前阵营产生根本分歧
- 重大事件：历史事件冲击导致价值观重新审视
- 人际影响：重要关系人的影响和劝说
- 利益驱动：新阵营提供更符合个人需求的发展空间
- 道德觉醒：通过人性测试获得新的道德认知

**转换后果**：
- 信任危机：原阵营成员的质疑和敌视
- 适应困难：新阵营文化和行为规范的学习
- 身份认同：在新旧身份间的心理调适
- 关系重构：人际关系网络的重新建立
- 道德负担：背叛标签带来的心理压力

#### 具体实现示例

**人格模板示例**：
```
理想主义型：
- 战斗被动："正义之怒"（对邪恶阵营+20%伤害）
- 社交特征：道德选择优先，难以妥协
- 阵营偏好：民间义士派、宗教超脱派
- 试探敏感度：中等（容易被道德话题影响）

现实主义型：
- 战斗被动："冷静分析"（免疫魅惑、混乱状态）
- 社交特征：利益导向，善于权衡
- 阵营偏好：地方豪强派、技术中立派
- 试探敏感度：高（能快速识别利益陷阱）
```

**人性测试事件示例**：
```
事件："饥荒中的选择"
触发条件：资源稀缺度>80%，角色处于领导地位
选项：
A. 优先分配给老弱病残（+理想主义，-实用主义）
B. 优先分配给壮年劳力（+实用主义，-理想主义）
C. 平均分配给所有人（+集体主义，-个人主义）
D. 优先保证自己家族（+个人主义，-集体主义）

后果影响：
- 道德声望变化：±10-30点
- 阵营倾向调整：±5-15点
- 触发后续事件概率变化
- 同伴忠诚度变化：±10-20点
```
