## 世界观
每个人、每样东西都像爬楼梯一样在“进化阶梯”上一步步变强、变聪明，但这楼梯不是直的，而是弯弯曲曲，受周围环境、和别人聊天打架，还有突然冒出来的事儿影响。每个家伙，从小村里的老农到大城市的国王，都有一本从生到死的完整“人生小说”。这些小说不是一个人关门写的，而是大家的故事你中有我、我中有你，织成一张大网。游戏里强调“无巧不成书”，意思是啥事儿是纯巧合，总有故事在背后推一把，让每次碰面或吵架都变成大戏的一部分。我们模拟从一个小村子到整个国家的生活，玩家和电脑控制的角色（NPC）干啥都能搅动大局。这个“进化阶梯”不光是变强那么简单，它代表人生就是不断适应的过程：你帮了别人，可能就解锁新技能；干坏事儿，可能会掉级。深刻来说，它外延到现实生活，提醒我们选择和互动塑造命运。外延开来，在游戏里，这理念延伸到社会层面——一个村庄的兴衰取决于居民的集体进化，国家强弱看民众的道德积累。甚至可以想成哲学：混沌中找秩序，人与人的连接是宇宙的本质。

## 现实设定
打个比方，这个游戏世界就像咱们现实的“老祖宗时代”，全都是架空的古代味儿，没啥高科技，就靠历史故事和幻想元素混在一起。时间过得慢吞吞的，现实1分钟等于游戏里1小时（1:60），所以游戏一天就是你现实24小时的浓缩版。这设计让玩家感觉日子过得真真切切，不赶趟。特别的地方有：点开菜单时间就停了，好让你慢慢琢磨；退出游戏，世界还自个儿转，确保你回来时一切都变了样，比如老朋友没了，新帝国冒头了。整个设定就是要制造一个“活蹦乱跳”的永恒世界，你不在，它也自个儿玩得欢。

系统在后台持续推进世界时间（即使玩家离线），每天根据节气、气候、历史发展等推进NPC事件。例如，离线时若是冬天，则NPC可能经历寒潮、迁徙、疾病；回归后通过“世界日志”告知玩家变化。菜单暂停功能限制为特定界面，防止时间滥用暂停。

### 借鉴设计理念
整合《Dwarf Fortress》、《Crusader Kings》、《RimWorld》和《模拟人生》的元素，聚焦核心系统。

**生存系统** (RimWorld & Dwarf Fortress):
- 设计: 角色管理饥饿、健康、情绪，通过采集和事件响应。
- 原则: 简化复杂模拟，随机事件添趣，避免负担。
- 落地: AI循环每tick检查需求，行为树驱动行动。

**家族与政治系统** (Crusader Kings):
- 设计: 传承机制，互动如联盟、冲突，以道德导向。
- 原则: 平衡深度，和谐结果，易上手。
- 落地: 事件链脚本，数据库追踪关系。

**个性与生活系统** (模拟人生):
- 设计: 独特trait、日常routine、情感互动。
- 原则: 强调个性，MOD友好扩展。
- 落地: 属性表和状态机，支持自定义模块。

整体原则: 深度与可玩性平衡，开发者易迭代。

## 历史
- **时代背景**：故事背景像架空的明朝，从老封建往早期工业变。皇帝是大老板，管着松散的国家联盟，但地方大佬和宗教势力总想抢风头。历史不是死的，会随玩家和NPC的举动变样。比如，你帮了个小村子，它可能变成大城市，改写历史书。 <mcreference link="https://www.dlyj.ac.cn/EN/10.11821/dlyj201810012" index="1">1</mcreference>
- **关键事件**：
  - **朝廷崛起**：一个英雄大佬统一东亚，建起帝国，推保守政策，强调道德稳定。开头繁荣，大家日子好，但后来腐败爬上来。比如，英雄叫李明，骑马打仗统一天下，建都城后发金币给大家庆祝，还修大运河方便贸易。 <mcreference link="https://www.progressingeography.com/EN/10.18306/dlkxjz.2017.02.006" index="2">2</mcreference>
  - **激进改革**：新势力带进火器和手工革新，引发内战。保守派死扛，激进派推工业，战争重画国家地图，还影响科技跳跃。比如，一场大战用新火炮炸开城墙，逼老皇帝退位，新领袖建工厂产枪炮。
  - **宗教战争**：老宗教派系打架，向善教义对上异端，搅动文化大变，诞生新信仰和英雄传说。比如，圣女领导起义，宣称新神迹，吸引百万信徒，战争中寺庙被毁又重建。
  - **近期事件**：玩家来前，一场大瘟疫加速大家进化，好多人得新技能，社会大变革，旧秩序摇晃。比如，瘟疫让农民学会种新作物，商人学会新贸易路线，工匠学会新工艺，军队学会新战术。
 
关键事件由“历史分岔点系统”驱动，玩家行为参与关键节点后，选择将影响国家/宗教/技术走向。系统存档所有重大节点，允许多线并行发展，不同角色走出不同世界史。

## 科技
- **水平**：科技处于轻火器时代，手工业高度发达，工业化初期。蒸汽动力初步实验，但依赖手工匠人。这时代科技不光是工具，更是人文进步的推手，通过人与人合作发明新玩意儿，推动社会从农业到工业变迁。
- **发明与应用**：
  - **武器**：传统剑、弓箭辅以火绳枪和早期炮弹。军队训练结合武术和火器战术，但竞争中，匠人们合作改进火器，导致战争更高效，也带来和平谈判的机会。
  - **交通**：马车主导陆路，帆船海运，初步铁路用于矿业运输。这些发明源于商人和工匠的合作，缩短距离，促进文化交流，但也引发资源争夺的衰退。
  - **日常工具**：发达纺织机、冶金炉，改善农业产量。医疗依赖草药，但工业化引入简单机械辅助。农民和医生合作，解锁新方法，提升生活质量，推动认知进步。
  - **进化影响**：科技与进化阶梯融合，如通过善行解锁发明灵感。竞争中，有人独占技术导致衰退，而合作分享则加速大家进步。
- **科技发展**：科技从火器航海到工业革命，社会从农业到工业，人口从几千万到亿级，城市冒头。比如，从泥巴村变高楼大厦，街上马车换成蒸汽车。这进步源于人与人致力于探索目标，不约而同造成社会跃进，但贪婪竞争也可能引发环境衰退。

玩家通过科技图谱系统推进科技研发。科技点来源于任务奖励、合作行为、文物探索等。科技进程可影响建造、新技能、战争单位解锁。NPC组织亦会自主研发，可能因理念不同与玩家科技冲突。

## 文化
- **道德体系**：文化以道德为基础，融合儒家思想和进化哲学。仁义、忠诚、和谐是价值观，社会规范鼓励通过善行提升地位。这体系服务人文发展，教人合作共赢，推动认知从个人到集体进步。
- **社会结构**：分层社会，从村庄长老到国家皇帝。家庭单位重要，教育强调道德故事。艺术形式如诗歌、绘画反映生命轨迹。在竞争中，阶层间合作（如长老和农民联手抗灾）带来繁荣，冲突则导致衰退。
- **习俗**：节日庆祝进化里程碑，婚礼和葬礼融入剧本事件。比如，中秋节大家聚会分享故事，促进合作；竞争习俗如比武大会，激发创新但也可能酿成仇恨，最终服务于整体人文进步。
- **文化动态**：人与人致力于艺术和教育目标，不约而同造成文化繁荣或衰落。比如，诗人合作编书启发后代认知，而文化战争则抹杀知识。

引入“文化动量”机制，不同区域文化影响力以动态值表现，随节日、活动、战争变动。文化冲突或融合将改变NPC思维模型，如一地儒家影响力下降，激进派思想上升，则NPC更倾向冒险行为。

## 宗教
- **核心信仰**：向善宗教崇拜“善之源”，相信行善提升进化阶梯。寺庙遍布，祭司指导道德决策。这信仰驱动人文发展，鼓励合作行善，推动认知从迷信到启蒙。
- **政体整合**：国家融合早期宗教，保守政体维护传统寺庙，激进政体实验新教义。保守派强调稳定合作，激进派竞争创新，导致宗教进步或派系衰退。
- **派系**：保守派强调稳定祭祀，激进派推动改革以加速进化。派系间竞争（如辩论大会）激发新思想，合作则融合理念，服务整体认知进步。
- **宗教影响**：信徒致力于不同目标，如保守者守旧、激进者革新，不约而同造成社会进步（如新寺庙建校）或衰退（如圣战破坏）。

玩家可加入或创立宗教，参与晋升、祭祀、圣战、传播等活动。宗教设有“信仰树”，解锁仪式、教义、建筑，影响社会结构与科技发展。例如玩家推动“科技即神恩”教义，则科技速度加快，但保守派NPC敌意上升。

## 生命轨迹与互动
每个角色有详细剧本：出生随机生成背景，成长学习技能，互动触发事件链，死亡影响世界。模拟从村庄经济到国家政治，动态沙盒允许玩家塑造叙事。

每名NPC由AI行为树控制，具备“记忆链条”系统，记录其过往遭遇与关键人物。系统按“命运感”算法生成其下阶段目标与冲突，如某人儿时被抢劫，则成年后可能成为守卫或盗贼。玩家行为将永久改变这些轨迹。


## 系统关联与整合

### 核心系统架构
游戏采用"角色生命周期"作为核心枢纽系统，所有其他系统都围绕这个中心进行整合。每个角色从出生到死亡的完整生命历程，成为连接各个子系统的主线。

### 系统间关联矩阵

#### 一级关联（直接整合）
- **角色生命周期 ↔ 所有系统**：作为核心枢纽，生命周期系统在每个阶段都与其他系统产生交互
  - 出生阶段：基因系统决定天赋，文化/宗教影响初始信仰
  - 成长阶段：个性系统塑造trait，教育事件影响各系统发展
  - 成年阶段：政治、科技、文化、宗教事件驱动社会化
  - 衰老阶段：生存系统debuff增加，各系统传承准备
  - 死亡阶段：所有系统的记录和影响传承给后代

#### 二级关联（深度耦合）
- **命运系统 ↔ 个性系统**：个性trait影响命运片段的生成概率，命运事件反过来调整情感状态
- **家族政治 ↔ 文化系统**：家族道德值受文化动量影响，政治事件推动文化变迁
- **宗教系统 ↔ 科技系统**：宗教派系影响科技发展方向，科技进步挑战传统信仰
- **生态系统 ↔ 生存系统**：生态资源循环直接影响角色生存需求，生存行为改变生态平衡
- **文化系统 ↔ 宗教系统**：文化道德体系与宗教信仰相互强化或冲突

#### 三级关联（间接影响）
- **科技 → 生态 → 生存**：科技进步改善生态产出，影响角色生存条件
- **宗教 → 文化 → 政治**：宗教理念塑造文化价值观，进而影响政治决策
- **个性 → 命运 → 家族**：个人性格影响命运走向，命运事件影响家族声望
- **生态 → 文化 → 科技**：生态变化推动文化适应，文化需求驱动科技创新

### 数据流向与同步机制

#### 事件传播链
1. **玩家行为** → 命运片段记录 → 影响附近NPC行为树 → 触发相关系统事件
2. **系统事件** → 更新角色传记 → 影响生命周期阶段 → 调整其他系统参数
3. **环境变化** → 生态系统响应 → 影响生存需求 → 驱动文化/科技适应

#### 传承机制整合
- **基因传承**：外貌、天赋、天命倾向
- **文化传承**：道德体系、习俗、动量值
- **知识传承**：科技进度、生态经验、生存技能
- **社会传承**：家族声望、政治关系、宗教影响力
- **个性传承**：trait倾向、情感模式、行为偏好

### 系统平衡与反馈

#### 正反馈循环
- 善行 → 道德提升 → 社会声望 → 更多合作机会 → 科技/文化进步
- 科技进步 → 生存改善 → 人口增长 → 文化繁荣 → 更多创新

#### 负反馈调节
- 过度开发 → 生态破坏 → 资源短缺 → 生存压力 → 社会冲突
- 宗教极端 → 文化冲突 → 政治动荡 → 科技停滞 → 社会衰退

#### 动态平衡机制
- **时间衰减**：所有系统的影响都有时间衰减，防止单一事件永久影响
- **随机扰动**：定期的随机事件打破系统平衡，创造新的发展机会
- **多元制衡**：不同系统间的相互制约，防止某个系统过度主导

### 技术实现要点

#### 数据库设计
- **中央事件表**：记录所有跨系统事件，便于追踪影响链
- **关联索引**：建立系统间的快速查询索引，提高响应效率
- **版本控制**：支持系统状态回滚，便于调试和平衡调整

#### 性能优化
- **分层更新**：高频系统（生存、个性）与低频系统（文化、宗教）分离更新
- **区域限制**：某些系统影响限制在地理区域内，减少全局计算
- **延迟计算**：非关键影响采用延迟计算，平衡实时性和性能
